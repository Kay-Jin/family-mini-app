# 数据库权限：示例与说明

微信云开发 **自定义安全规则** 因环境、版本差异较大，请勿在未验证的环境下直接用于生产。

本项目仍以 **`docs/cloud-database-rules.md`** 为主文档；此处仅作索引。

建议步骤：

1. 在控制台「权限设置」中使用官方 **规则校验器**。
2. 开发阶段优先 **禁止客户端写**，业务写走 **云函数**。
3. 如需「按 household 读」，再逐步引入带 `auth.openid` 条件的规则，并在真机验证延迟与误拒。

若你有可用的脱敏规则片段，可经评审后追加到本目录（`.json`），但需标注测试环境与适用 SDK 版本。

# 合规与数据主体权利（提纲）

> 非法律意见；上线前请由法务/隐私顾问审阅。

## 建议产品侧明确告知用户

- 收集的字段：`openid`、家庭关系、`householdId`、昵称、健康与报平安等业务数据用途与留存周期。
- 家庭场景：**管理员/创建者** 可见范围与其他成员权限。

## 用户请求「删除账号 / 导出数据」

1. **导出**：提供人工或自动导出 `members` + `household_members` 中关联记录的流程说明（联系方式、处理时限）。
2. **删除**：在验证身份后，**软删 + 异步清理**云库文档（注意 `household_members`、`invite_codes`、子集合引用一致性）。
3. **审计日志** `family_audit_logs`：评估是否纳入删除范围或单独保留策略。

## 未成年人

若面向未成年人，需根据适用法域补充监护人同意与适龄设计（当前代码未特化）。

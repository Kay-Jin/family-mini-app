# 晨报订阅消息（MVP 接入说明）

小程序侧已实现：

- **「我的 → 接口配置」**：填写 **晨报订阅模板 ID**（微信公众平台 → 功能 → 订阅消息），**逗号分隔，最多 3 个**，与「保存接口配置」一并写入本地存储。
- **「今日」→ 晨报卡片**：按钮 **「订阅晨报提醒」** 调用 `wx.requestSubscribeMessage`。

## 你仍需在公众平台完成

1. 申请/选用与 **晨报** 文案匹配的 **订阅消息模板**（类目需符合审核要求）。
2. 将模板 ID 复制到小程序配置输入框。
3. **云函数定时推送**：用云开发 **定时触发器** 调云函数，在业务逻辑里检查用户是否已授权对应模板（微信会返回用户订阅状态 scope），再 **`subscribeMessage.send`**（需按官方文档传入 `page`、`data` 等）。

> 当前仓库 **未** 内置定时器与 `send` 实现，避免误发；上线前在云函数侧补齐并与 `morning_briefs` 数据源联动。

详见微信文档：订阅消息、一次性订阅、`wx.requestSubscribeMessage` 行为差异。
